This plugin allows website owners to show a search box widget, to let their users find parking spaces on https://parku.com.

== Description ==

## parku Search  Widget
With this widget it is possible to show a customizable parku search in any widget area on your website. It allows users to easily find a parking space with parku.

This version supports the following locales:
* en_GB
* de_DE
* fr_FR
* nl_NL

It is possible to change the labels of the form in the wp-admin widget area to any language you want.

== Installation ==

1. Install & activate this plugin through wp-admin
1. Drag the 'parku Search' widget to a widget area in the wp-admin 'Widgets' section
